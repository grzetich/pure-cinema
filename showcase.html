Install

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for "Pure Cinema"
  4. Click Install

Quick Start

1. Ctrl+Shift+P ‚Üí "Pure Cinema: Start Recording" 2. Use your terminal normally 3. Ctrl+Shift+P ‚Üí "Pure Cinema: Stop Recording" 4. Save your .pcr file

Commands
