How It Works

1

Start Recording

Open command palette in VS Code and run "Pure Cinema: Start Recording". A new terminal opens ready to capture your session.

2

Run Commands

Execute your commands normally. Pure Cinema captures everything - input, output, timing, and colors in real-time.

3

Stop & Save

When done, run "Pure Cinema: Stop Recording". Your session is saved as a .pcr file with all interaction data.

4

Share Anywhere

Play back recordings in VS Code, export to HTML, or share directly to GitHub, LinkedIn, and other platforms.
